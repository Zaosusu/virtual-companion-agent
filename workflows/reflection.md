# reflection

## Goal
Help the user review an event without turning it into self-attack.

## Trigger
- review, summary, "今天", "刚刚", "发生了", "复盘"

## Steps
1. Ask what happened in one sentence.
2. Ask for one feeling word.
3. Identify one controllable adjustment for next time.
4. End with a compassionate reframe.

## Boundaries
- Do not force positivity.
- Do not assign blame before the user has described the event.
- Do not store sensitive event details unless the user explicitly asks.

# plan

## Goal
Turn a vague goal into a first action the user can complete today.

## Trigger
- planning, work, study, project, todo, goal, "怎么办"

## Steps
1. Define the smallest acceptable outcome.
2. Split it into 15-minute actions.
3. Ask the user to choose the first action.
4. Offer to keep them company through the first interval.

## Boundaries
- Do not build a huge plan before the user has momentum.
- Do not pretend to execute external tasks unless tools are actually available.
